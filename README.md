# Clone do Instagram (HTML e CSS)

Este projeto é um clone simples da interface do Instagram, desenvolvido utilizando apenas **HTML** e **CSS**.  
Ele foi criado como parte de um projeto da **DIO (Digital Innovation One)** com o objetivo de praticar conceitos básicos de desenvolvimento front-end.

O foco do projeto é exclusivamente educacional, não possuindo qualquer funcionalidade dinâmica, integração com backend ou manipulação de dados reais.

---

## Funcionalidades
- Layout estático inspirado no Instagram  
- Tela de login simulada  
- Estrutura visual semelhante ao feed  
- Noções básicas de responsividade com CSS  

---

## Tecnologias Utilizadas
- HTML5  
- CSS3 (Flexbox)  

---

## Como Executar o Projeto
1. Clone este repositório  
2. Abra o arquivo `index.html` em qualquer navegador  

Não é necessário instalar dependências adicionais.

---

## Objetivo do Projeto
Este projeto tem como finalidade:
- Praticar a estruturação semântica com HTML  
- Aprimorar o uso de CSS para layout  
- Desenvolver atenção aos detalhes visuais de interfaces
